# Sistema de Cadastro em Java

## Sobre o projeto
Sistema CRUD desenvolvido em Java com foco em:
- Programação Orientada a Objetos
- Validação de dados
- Tratamento de exceções
- Organização em camadas
- Manipulação de coleções com HashMap

---

## Funcionalidades
- Cadastro de pessoas
- Busca por CPF
- Remoção de usuários
- Atualização dinâmica de dados
- Validação de CPF
- Validação de Gmail
- Validação de telefone
- Ordenação alfabética

---

## Tecnologias utilizadas
- Java
- HashMap
- Streams API
- Exception Handling
- IntelliJ IDEA
